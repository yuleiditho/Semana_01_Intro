# Wireframe UI

Se diseño la interfaz de la aplicación de **Abogabot** desde la vista en un celular

* Página de inicio
* Crear una cuenta nueva
* Login
* Interfaz del usuario al iniciar sesión
* Simulación del chat con el bot
